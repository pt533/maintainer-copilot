# Code of Conduct

## Our standards

We want this project to be welcoming, respectful, and constructive.

Examples of positive behavior:
- being respectful in discussions
- giving actionable feedback
- assuming good intent
- focusing on the problem, not the person

Unacceptable behavior includes:
- harassment or personal attacks
- hateful or discriminatory language
- doxxing or sharing private information
- bad-faith disruption of discussions

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please open a private security-style report if safety is involved, or contact the maintainer through GitHub.

## Scope

This code of conduct applies within project spaces, including issues, pull requests, and discussions related to this repository.
